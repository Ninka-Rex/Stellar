## 

```
**Did the AI write bad code?**
├── No → Chill out
└── Yes
    └── **Can YOU fix it?**
        ├── Yes → submit a PR
        └── No
            └── **Will someone else fix it?**
                ├── Yes → open an issue and wait
                └── No
                    └── **Is there a human-written alternative?**
                        ├── Yes → migrate and move on
                        └── No → someone is trying their best, be grateful
```
